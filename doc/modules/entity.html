Module Entity

Everything in a game is represented by an Entity.

Oddly enough, entities have no state or logic.

Usage:

    local e = L.Entity()

Functions

Entity:addComponent (component) Will attach a component to an entity.
Entity:getComponent (klass) Will fetch an attached component (or return nil if not present).
Entity:eachComponent (fn) Will invoke the callback function with each component attached to this entity
