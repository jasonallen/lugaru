Module scene_mgr

Manages scenes and their ordering Currently just does Markdown, but this is intended to be the general module for managing other formats as well.

Functions

SceneMgr:add (scene)
Adds an extra scene to get rendered. Scenes are rendered in their priority order. Scenes are also accessible by their name.

Parameters:

  • scene The scene to be added

Usage:

  • local fooScene = FooScene("foo")
  • L.sceneMgr.add(fooScene)
  • ...
  • local scene = L.sceneMgr.scenes["foo"] -- retrieves the fooScene!
SceneMgr:remove (scene)
Define special sequences of characters. For each pair (find, subs), the function will create a field named with find which has the value of subs. It also creates an index for the table, according to the order of insertion.

Parameters:

  • scene The replacement pattern.
