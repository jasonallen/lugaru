Module Entity

Everything in a game is represented by an Entity.

Oddly enough, entities have no state or logic.

Functions

Entity:addComponent (component)
Will attach a component to an entity.

Parameters:

  • component: The component being added to the entity

Usage:

    local e = L.Entity()
local p = L.Position(0, 0)
e.addComponent(p)
