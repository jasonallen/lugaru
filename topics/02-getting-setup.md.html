Topic 02-getting-setup.md

Getting Setup

Before we start coding, you need to make sure you have a working environment. We're going to build this app for PC/Mac/Linux first - then we'll port it to iOS/Android after. There are three main requirements to have before we start:

  1. Moai: you need to have a moai host runner for your OS. It's going to be the program we invoke to run our game. We'll run moai and pass it our lua code: that's how it will run our game. We can embed our lua code into a single executable later - but this way we'll have the fastest code - run - debug cycle possible. To setup your Moai host:
  2. Download the latest Moai SDK (from Tarball or Github ).
  3. Use the appropriate direcoty to invoke your platform-specific IDE/Compiler (XCode for OSX, vs20x0/Visual Studio for Windows, cmake for linux). The goal is to generate the "moai" host app.
  4. Once you've built your "moai" host app add it to your path. Invoking it should emit something like this:
 $ moai
 MOAI-OPEN DEBUG
 Initializing Chipmunk v5.3.4 (Debug Enabled)
 Compile with -DNDEBUG defined to disable debug mode and runtime assertion checks

2. Start your new project 1. Create a new project directory - call it lupac (for Lua Pacman - clever, right?) 2. Inside your lupac directory create a subdirectory called "src" and "img" 3. Copy the lugaru src directory inside of your lupac/src directory and rename it "lugaru", so it ends up looking like this

 lupac/
   img/
   src/
     lugaru/
       <.. all the lugaru src code ..>

4. Copy this file http://2.bp.blogspot.com/Gm1MOUisRyY/TAXOOglk8I/AAAAAAAAuO4/2zEOzvaRTwk/s400/pac-man-logo.png to your img/ subdirectory. Make sure it's called pac-man-logo.png! 4. Create a file in your src/ directory called main.lua, and paste the following code:

 package.path = package.path .. ";src/?.lua;src/lugaru/?.lua"

 local L = require 'lugaru.lugaru'
 L.init(672, 864)

 local setup = function(scene)
   local e = L.Entity()
   e:addComponent(L.Position(0,0))
   e:addComponent(L.Sprite("img/pac-man-logo.png", 400, 85))
   scene:addEntity(e)
 end
